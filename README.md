# ColorMart - Premium Beauty & Cosmetics E-commerce Store

## 🚀 Live Demo
[colormart.store](https://colormart.store)

## 📋 Overview
ColorMart is a full-featured e-commerce platform for beauty and cosmetic products, built with modern web technologies and Firebase backend.

## ✨ Features

### Customer Features
- 🛍️ Browse products by categories
- 🔍 Advanced product search and filtering
- ⭐ Real customer reviews and ratings
- 🛒 Shopping cart with quantity management
- 💳 Secure checkout process
- 📧 Email notifications for orders
- 📱 Fully responsive design
- 🎨 Smooth animations with AOS

### Admin Features
- 🔐 Secure admin authentication
- 📦 Product management (add/edit/delete)
- 🖼️ Image compression (auto-resize to <500KB)
- ⭐ Review moderation
- 📞 Contact form management
- 📊 Dashboard with analytics
- 📝 Order management system

## 🛠️ Technology Stack

### Frontend
- HTML5, CSS3, JavaScript (ES6+)
- Vite - Build tool
- AOS - Scroll animations
- EmailJS - Email notifications

### Backend & Database
- Firebase Firestore (4 databases)
  - Main DB - User management
  - Products DB - Product catalog
  - Reviews DB - Customer reviews
  - Contacts DB - Contact submissions

### Hosting & Deployment
- Vercel - Hosting platform
- GitHub - Version control
